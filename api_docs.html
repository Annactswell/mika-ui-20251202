Mika API Reference

Build the next generation of creative apps with the Mika API. Generate art, upscale images, and unleash the monster within your code.

Base URL

https://api.mika.ai/v1

01 Authentication

Authenticate your requests using the Authorization header with your API key. 

curl -X POST https://api.mika.ai/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ ... }'

POST

/generate

Create a new image generation task.

Parameter Type Description
prompt string Description of the image to generate.
model string Model ID (e.g., "mika-v5", "sdxl").
aspect_ratio string "1:1", "16:9", "9:16". Default: "1:1".

Response

{
  "id": "task_123456789",
  "status": "processing",
  "created_at": "2023-10-27T10:00:00Z",
  "eta": 12.5
}
GET

/tasks/{id}

Check the status of a generation task.

Response (Completed)

{
  "id": "task_123456789",
  "status": "completed",
  "output": [
    "https://cdn.mika.ai/results/img_001.png",
    "https://cdn.mika.ai/results/img_002.png"
  ],
  "meta": {
    "seed": 4829103,
    "model": "mika-v5"
  }
}
POST

/upscale

Upscale an existing image to higher resolution.

Parameter Type Description
image_url string URL of the source image.
scale integer Scale factor (2 or 4). Default: 2.

Response

{
  "id": "task_upscale_987",
  "status": "processing",
  "eta": 5.0
}

Errors

Mika uses standard HTTP status codes to indicate success or failure.

400 Bad Request

Invalid parameters or malformed JSON.

401 Unauthorized

Missing or invalid API key.

429 Too Many Requests

Rate limit exceeded.

500 Server Error

Something went wrong on our end.
